# BugLens: A Hybrid AI Based Software Bug Management System

BugLens is an AI-powered bug detection and analysis system designed to identify potential bugs and vulnerabilities in code efficiently. The project leverages intelligent analysis techniques to help developers detect issues early, improve code quality, and reduce debugging time. It provides a user-friendly interface to analyze code and view results clearly, making it suitable for both learning and practical use.
